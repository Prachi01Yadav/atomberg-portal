Dry-run checklist

  • Before judges: API responds at /health; UI loads; run demo-prep.ps1 if goal submit fails (clears stray drafts).
  • Employee: My Goals → weightage 100% → Submit all → message or validation panel if blocked.
  • Manager: Team Dashboard → Approve employee goals.
  • Employee: Check-ins → quarter banner green/open (or Admin toggles “force window open”).
  • Admin: Analytics → Manager effectiveness; Audit log sample row.
